#### Install

`[sudo] pip install Desktop-command`

`[sudo] sudo npm install -g Desktop-command`

#### Usage
```bash
# desktop --help
usage: desktop COMMAND [options]

Available commands:
	hide		hide Desktop
	show		show Desktop
	toggle		hide/show Desktop
	wallpaper	get/set Desktop wallpaper
```
